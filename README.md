# 🌞 Site da Pré-Incubação | Vale do Sol - Assaí/PR

Este projeto é um site institucional desenvolvido para apresentar o programa de **Pré-Incubação de Projetos Inovadores** da cidade de **Assaí - PR**, celebrando os 93 anos do município com foco em tecnologia, inovação e desenvolvimento sustentável.

## 🚀 Objetivo

O objetivo deste site é divulgar as iniciativas de inovação da Secretaria de Ciência, Tecnologia e Inovação de Assaí (SECTI), facilitar a submissão de projetos por parte de cidadãos e instituições, além de apresentar informações sobre o programa de pré-incubação, eventos e resultados.


---

## 🛠️ Tecnologias Utilizadas

- **HTML5** semântico
- **CSS3** com design responsivo
- **JavaScript Vanilla** para interações
- **ScrollReveal.js** para animações on-scroll
- **Schema.org (JSON-LD)** para SEO e metadados institucionais
- **Content Security Policy (CSP)** para segurança

---

## 🔍 Funcionalidades

- Menu responsivo com modo mobile/hambúrguer
- Animações suaves com ScrollReveal
- Lightbox de imagens para galeria e prêmios
- Submissão de projetos via formulário
- Links para portais oficiais de Assaí e eventos comemorativos

---

## 🌐 SEO e Acessibilidade

- Uso de **metadados estruturados (Schema.org)**
- Textos alternativos (`alt`) nas imagens
- Boa hierarquia de títulos e navegação com `aria-*`
- Compatível com dispositivos móveis

---

## 📸 Destaques Visuais

- Paleta de cores inspirada na identidade do Vale do Sol (azul e amarelo)
- Cards animados com `hover` e `scale`
- Design leve e institucional, com foco em clareza e navegabilidade
